## 📖 Overview

**Web_Penetration_Testing_Framework** is an **all-in-one Web Penetration Testing Framework** developed by **Team Hackethics138**. It brings together **40+ powerful security tools** into a single, intuitive interface with **auto‑installation**, **real‑time output**, and **interactive prompts**. Whether you're a seasoned penetration tester or a beginner, WebPWN simplifies the process of discovering and exploiting web vulnerabilities.

> **⚡ Note:** This tool is intended for **authorized security testing only**. Unauthorised use is illegal and unethical.

---

## 🔥 Key Features

| Category | Tools & Capabilities |
|----------|-----------------------|
| 🕵️ **Reconnaissance** | OSINT, subdomain enumeration, port scanning, DNS reconnaissance (nmap, theHarvester, sublist3r, amass, nuclei, etc.) |
| 🧩 **Scanning & Enumeration** | Web server scanning, CMS fingerprinting, WAF detection (nikto, wpscan, joomscan, wafw00f, whatweb) |
| 💉 **SQL Injection** | Automated and manual SQL injection testing (sqlmap, custom payloads) |
| 🕸️ **XSS Attacks** | Advanced XSS detection and exploitation (XSStrike, Dalfox, xsser) |
| 🔑 **Authentication Attacks** | Brute‑force, password cracking, JWT testing (hydra, john, hashcat, jwt_tool) |
| 📂 **LFI / RFI** | Local/Remote file inclusion exploitation (liffy, kadimus, fimap) |
| 💻 **Command Injection** | OS command injection and Shellshock scanning (commix, shellshock‑scanner) |
| 🌐 **SSRF** | Server‑Side Request Forgery exploitation (SSRFmap, Gopherus, Interactsh) |
| 📦 **CMS Exploitation** | WordPress, Joomla, Drupal vulnerability scanning (wpscan, joomscan, droopescan) |
| 🧰 **Miscellaneous** | Metasploit, Burp Suite, ZAP, Netcat, WFuzz, SearchSploit, and more. |

✅ **Auto‑Install** – If a tool is missing, the framework asks for permission and installs it automatically (both on Kali and Termux).  
✅ **Real‑time Output** – See results as they happen – no waiting for full scans to finish.  
✅ **Interactive Prompts** – You're guided through each step: target URL, wordlist selection, custom parameters.  
✅ **Beautiful Interface** – Colored banner, menus, and status indicators (exactly like your screenshot).  
✅ **Cross‑Platform** – Works seamlessly on Kali Linux, Ubuntu, and even Termux on Android.
